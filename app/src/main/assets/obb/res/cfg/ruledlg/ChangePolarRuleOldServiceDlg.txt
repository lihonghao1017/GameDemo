h2 老服限制
# YYYY-MM-DD HH:MM 后（开服1年后，以角色创建或转换功能推出时所在区组的开服时间 YYYY1-MM1-DD1 为准），等级达到70级的道友可脱离自身所属门派，转换至其他任意门派，并可以再次重返原师门。
# 每位道友每年只有1次转换次数，跨年后(1月1日后)重新获得一次转换次数，未使用的次数不会累计。
# 转换门派的花费为17600金元宝起，自身道行越高费用越高，重返师门需固定耗费17600金元宝。
# 转换时不可携带武器，装备亦不会一同转换。
# 转换时不会重置属性点和相性点分配。
# 每位道友只能拥有1种门派的万道归一称谓，转换门派前若已拥有该称谓，则转换后将被回收，且在新门派不会触发【万道】师尊传召任务。
# 重返师门或转回原门派时，会获得曾经拥有的万道归一称谓。
# 转换门派成功后30日内可领取重返师门任务，超出后将不可再领取。
# 重返师门有效期内，若没有转回原门派，暂时无法在聚宝斋交易角色。(仅官方渠道显示本句)
# 重返师门有效期内，转回原门派或超出有效期，可以继续在聚宝斋交易角色。(仅官方渠道显示本句)
h2 新服限制
# YYYY-MM-DD HH:MM 之前（开服90天内），等级达到70级的道友可脱离自身所属门派，转换至其他任意门派，并可以再次重返原师门。
# 每位道友只有1次转换门派和1次重返师门的机会。
# 转换门派需耗费2张改头换面卡，重返师门需耗费1张。
# 转换时不可携带武器，装备亦不会一同转换。
# 转换时不会重置属性点和相性点分配。
h2 其他说明
# 新服转换次数与老服转换次数独立，不会相互占用。
