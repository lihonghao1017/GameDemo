D1
h1 保卫雪域
h2. 背景故事
# 极寒雪域遭到了火怪入侵，雪域的居民却一盘散沙。把他们团结起来并打败火怪吧！
h2. 招募居民
# 通过巡逻可遇到雪域居民。
# 在战斗中战胜居民及其助手冰龙，或成功捕捉居民，即可招募其加入队伍。
# 捕捉时，对应居民血量越少，成功率越高。
# 居民分为普通、优良、卓越3种品质，品质越高属性越高，打火怪时战斗越轻松。
h2. 遣散居民
# 最多可与4名居民并肩作战，超过时若遇到新的居民，则需要遣散已有居民。
# 在组队界面点击居民即可进行遣散操作。
h2. 挑战首领
# 成功招募居民后，即可前往火怪巢穴挑战火怪首领。
# 战胜火怪首领即可成功，挑战失败后仍可再次挑战。
# 战斗中可长按怪物设置集火目标。
