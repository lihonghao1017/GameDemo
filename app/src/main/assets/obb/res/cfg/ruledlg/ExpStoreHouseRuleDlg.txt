D1 规则说明
h2 开启条件
# 等级达到70级且不高于&1级的角色锁定经验7天以上即可开启经验仓库系统。
h2 经验存储
# 系统开启期间在真身状态下完成日常活动所获得的经验将被储存。
# 日常活动包括：师门、修炼、悬赏、除暴、通天塔、副本、助人为乐任务与使用修行、修炼卷轴。
h2 经验领取
# 解除经验锁后进入领取期，期间可领取之前存储的经验，快速升级！
# 只有处于真身状态才可领取经验。
# 每天可领取1次，每次可领取总量的1/7的经验，最快7天可领取完所有经验。
# 若角色真身等级比本区组最高真身等级低1级则无法继续领取经验。
# 若领取经验后角色真身等级将达到本区组最高等级则无法领取经验。
# 领取期需要在14天内将经验领取完，否则储存的经验将清空。
# 领取期期间若又重新锁定经验，则储存的经验将清空。
# 储存的经验值上限为&2亿（初始为20亿，开服时间超过365天后根据开服时长、角色等级获得额外的上限值）。
