D1 
h2 购买仙班
# 获得位列仙班或购买更高级别的位列仙班可更换排队通道，节约排队时间。
# 购买成功后可使用当前账号下任意角色进入游戏领取，但仅有1名角色可领取。
# 领取有效期为自开服之日起30天内，超出期限后仙班资格将被系统回收。
# 本次新服开启位列仙班打折活动，在本界面时购买时仍需支付原价，进入游戏后领取时，将会返还折扣元宝。
# 本次新服商城出售位列仙班礼包，在本界面购买位列仙班，进入游戏后领取时，将可领取到对应级别的位列仙班礼包。
# 本次新服商城出售位列仙班·高托礼包，在本界面购买位列仙班，进入游戏后领取时，将可领取到对应级别的位列仙班·高托礼包。
# 本次新服开启充值好礼双倍活动，进入游戏后领取时，将可获得额外赠送的充值好礼次数。
# 本次新服开启充值积分活动，进入游戏后领取时，将可获得充值积分，可前往福利→充值积分界面兑换奖品。
# 本次新服开启消费积分活动，进入游戏后领取时，将可获得消费积分，可前往福利→消费积分界面兑换奖品。
# 本次新服开启新年祈福活动，进入游戏后领取时，将可获得祈福券，可前往福利→新年祈福界面祈福领奖。
