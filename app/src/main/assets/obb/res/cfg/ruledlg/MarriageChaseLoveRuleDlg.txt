h2 基础规则
# 游戏共计3关，只能单人参与。
# 点击开始行动按钮可正式开始游戏，在开始前可预先调整箭头方向。
# 方向箭头及固定箭头都可改变主角及其所乘船只的移动方向，主角以外的人物、船只都不受箭头影响。
# 以到达每关终点为目标来调整箭头，途径特殊格子会触发各类事件。
#i Qianlxh/QianlxhRulepic.png,Qianlxh.plist
h2 注意事项
# 人走入水中、小舟撞击礁石或其它船只（小舟、战船）、误入漩涡等都会导致挑战失败。
# 挑战失败后，系统会自动将关卡和剧本重置为初始状态，并可以重新开始。
h2 操作规则
# 改变格子上的箭头方向，使主角根据箭头指引进行移动。
# 只能点击方向箭头，固定箭头不可点击。
# 进入船只所在格子即可使船只移动。
