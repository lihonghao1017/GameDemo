h1 规则说明
# 活动期间等级到达70级方可获得新服盛典排名评比资格。
# 排名以本次新服盛典活动冲榜阶段结算时的数据为准。
# 排名结算后的当天，系统将从20:30:00开始第一轮开奖，每5分钟开奖一次，直至所有6个大奖开奖完毕。开始开奖时，名单会持续滚动一定时间，并最终停止在大奖得主的名字上。
# 开奖环节将分别从等级榜、道行榜排名第1~1000、1001~3000、3001~6000名的道友中各抽取1名获得神兽大奖、变异大奖、六阶精怪大奖，共计抽取6名大奖得主。
# 开奖期间，每名道友最多只会被开中一次大奖（即神兽、变异、精怪奖励）。
# 开奖结束后，等级榜、道行榜排名第1~1000、1001~3000、3001~6000名中未抽中大奖的道友将分别获得巅峰盛典礼包、高级盛典礼包、中级盛典礼包。
# 未进入榜单的道友，在开奖结束时如果等级达到70级，将获得初级盛典礼包。
# 若是双榜都有名次，则抽中大奖后，未中奖的排行榜只会发放对应名次的盛典礼包。
