D1
h1 禁火寒食规则
h2 基础规则
# 活动期间，每条线的天墉城都会出现一定数量的#Y火妖#n。
# 帮助#Y望火楼武侯#n消灭火妖（火妖少于一定数量时，将会在天墉城中随机刷新）。
h2 消灭火妖
# 对付火妖前需在#R天墉城南边#n的荷塘附近打水。
# 向火妖浇水灭火，并在战斗中对火妖本体使用【提桶灌顶】即可消灭火妖。
# 每条线消灭一定数量的火妖后，将会在天墉城触发下雨天气。
# 在与火妖战斗中战败时，会被火妖诅咒（即变身为火妖），触摸活动大使附近的#Y冰晶石#n即可解除诅咒。
h2 奖励说明
# 消灭5只火妖即可在领奖时获得5粒青团。
# 每日最多可获得并食用5粒青团。
# 青团3日后将会超出保质期（即该道具限时3天），请及时食用。
# 在#Y望火楼武侯#n处领取奖励时，将获得随机道具奖励。
h2 青团作用
# 食用青团可获得经验、道行。
