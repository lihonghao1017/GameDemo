D1 
h1 帮派红包
h2 开启条件
# 帮派人数达到20人，可开启帮派发红包功能。
# 帮派成员需达到50级，方可使用帮派发红包功能。
h2 发放规则
# 可发放红包的活动包括：培育巨兽、挑战巨兽、帮派智多星、帮派温泉、帮派宴会、混元仙树、帮派焰火、帮派踩方块。
# 每日可发送各类型帮派红包共计8次，次数每日05:00:00更新。
# 24小时内开启且参与人数≥10人的活动，在活动时间结束时，可发放对应活动红包。
# 功臣红包仅在本周有功臣时由帮主或副帮主发送。
# 帮战或跨服帮战结束后24小时内，帮主或副帮主可发送帮战红包。
h2 领取规则
# 只有参与指定活动的帮派成员才能领取活动红包。
# 每名功臣每周限领1次功臣红包。
# 需参与帮战或跨服帮战才能领取帮战红包。
h2 其他须知
# 红包类型及对应的成员数量会影响红包的金额范围，金额不同红包可拆分的份数也不同。
# 发放的红包将在帮派福利—帮派红包界面、帮派聊天界面中刷新。
# 通过红包领取的元宝将转换为银元宝。
# 红包保存6小时，超过6小时后未被领取元宝将以邮件返还。
