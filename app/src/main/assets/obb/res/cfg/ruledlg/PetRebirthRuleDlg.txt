D1
h1 宠物重生
h2 变化部分
# 宠物等级降至70级，四项属性基础数值均降至70点，重置属性点、抗性点为未分配状态。
# 宠物技能等级上限回退至112级，重生前未超出等级上限的技能不会变化。
# 宠物若已飞升则回退至未飞升状态，成长资质回退至飞升前的数值。
# 清零【武学】与【亲密度】数值，所清零亲密度数值不包括天书加成部分。
# 清除全部妖石，取消所有天书启用状态。
h2 保留部分
# 保留已有成长洗炼、天技洗炼、强化、点化、羽化、幻化、进化、培元、凝神等养成。
h2 其他说明
# 神兽、鬼仙与8阶及以上的坐骑重生价格高于其他类型宠物。
# 重生正在进行宠物探索的宠物时，不影响该次探索的结果。
