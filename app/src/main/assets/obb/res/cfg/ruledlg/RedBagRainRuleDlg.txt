h1 如意红包雨
h2 基础规则
# 活动期间每天12:00、22:00，系统会在天墉城发放大量红包，10级以上的玩家均可参与抢夺。
# 届时主界面将提前出现红包按钮，点击后可快速进入抢红包界面。
# 每个时间段将降下5波红包雨，每波持续10秒，期间可随意点击开抢，数量不限。
h2 参与条件
# 只有在天墉城才可参与红包雨活动，请道友提前返回做好准备。
# 请在开抢前退出自动任务状态(如自动刷道、自动除暴等)。
# 战斗状态、非暂离队员不可参与。
h2 红包奖励
# 每25个红包可折算成1张兑换券，抢红包时还有几率获得珍稀的召唤令·十二生肖。
# 使用兑换券可以在兑换商店中兑换各种游戏道具。
# 奖励多少，全凭手速，大家快来参与吧！
