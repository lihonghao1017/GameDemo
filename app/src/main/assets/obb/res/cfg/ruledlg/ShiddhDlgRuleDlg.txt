D1
h1 试道大会
# 试道大会乃切磋论道之盛事，于每周日21:10开启，等级至少达到60级的道友方可参与，开启前10分钟为报名和入场时间，入场的队员与队长需处于同一组别且入场后不可再调整队伍，请至少组满3人后入场，开启时场内人数需至少5人方可开启成功。
# 每单数月第二个周三前的周日和单数月第二个周三当周的周日将举行月道行试道大会，在月道行试道大会中，所有道友战斗中道行将根据上月道行计算效果，宠物武学将根据上月武学计算效果。
# 试道大会分为挑战元魔和巅峰对决两个阶段。挑战元魔阶段道友打败试道元魔会获得大量经验、道行奖励及元魔积分，挑战元魔结束后根据队伍元魔积分（队员元魔积分之和）排名挑选一定数量的道友获得进入巅峰对决的资格，每次挑选的数量与该等级段上一届的参与人数有关，最多不会超过900人。
# 巅峰对决为PK阶段，由系统自动分配对手，场内最后一支队伍或场内积分最高的队伍将成为试道王者队伍。
# 试道王者队伍的所有道友将会获得特殊称谓、商城道具等丰厚奖励。道友在巅峰对决阶段的战斗中死亡次数超过对决值（元魔积分数量/ 20+1）时将被送出试道场但不会有任何损失。
# 活动结束后，如果决出了试道王者队伍，则从剩余有巅峰对决资格的队伍中选择队伍试道总积分第一至第九的队伍获得试道第二至第十名，排名前十的队伍也可以获得特殊称谓、商城道具奖励。
# 排名时若出现积分相同的情况，队伍成员道行之和高的排名靠前。
# 排名奖励领取截止时间为下一周周日04:59，超出后将不可再领取。
