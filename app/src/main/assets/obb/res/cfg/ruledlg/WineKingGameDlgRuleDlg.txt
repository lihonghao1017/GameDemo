D1
h1 酒道大会
h2 游戏目标
# 不断喝酒增加分数，倒计时结束时得分最高者获胜。
h2 基本操作
# 点击右下角【喝酒】按钮，将绿点停在判定条上的任意橙色区域即可成功喝酒。
h2 喝酒
# 绿点停留在【喝尽】对应的橙色区域（最窄），可以直接喝完100%的酒，获得100分，并大幅提升醉酒值。
# 绿点停留在【一半】对应的橙色区域（中等宽度），可以直接喝完50%的酒，获得50分，并中幅提升醉酒值。
# 绿点停留在【小口】对应的橙色区域（最宽），可以直接喝完25%的酒，获得25分，并小幅提升醉酒值。
# 酒水比例为0%时，会自动更换一坛酒。
h2 醉酒
# 醉酒值越高，绿点移动速度越快。
# 醉酒状态达到【状若癫狂】时，无法继续喝酒，必须通过解酒才可缓解。
h2 解酒
# 点击【解酒】按钮，可缓解醉酒状态。
# 使用【解酒】后5秒内不可再次使用，每15秒恢复1次。
