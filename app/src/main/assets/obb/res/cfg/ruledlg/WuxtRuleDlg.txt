D1
h2 修炼条件
# 70级以上的宠物可进入武学台修炼或担任武学教头。
# 修炼时必须有武学教头带领才可进行修炼。
# 修炼会消耗武学台的耐久度，若耐久度不足则无法进行修炼。
# 教头只会教授武学给符合要求的宠物（要求详见界面提示）。
h2 身份待遇
# 不同等级段的武学教头可给弟子带来不同的身份待遇。
# 教头的等级段共三档：【70-94】级、【95-134】级、≥135级，分别对应弟子身份：普通弟子、核心弟子、嫡传弟子。
# 等级段越高则修炼效率越高，消耗也越高。
# 弟子等级必须与教头处于同一等级段且与教头的等级差需≤5级。
# 宠物等级按照开始修炼时的等级计算，修炼期间提升等级将不影响修炼效率。
h2 奖励说明
# 修炼时可产生武学，需携带对应宠物返回居所才可领取。
# 可消耗元宝给武学台补充醍醐值，醍醐值可用于醍醐灌顶。
# 武学产出每10分钟结算一次，醍醐在结算武学产出时随之消耗并使宠物额外获得1倍武学奖励。
h2 注意事项
# 每个道友只可拥有一座武学台，每座武学台最多可供2只宠物修炼。
# 在武学台教学、修炼的宠物仍可参战、掠阵、骑乘。
# 可通过点击宠物形象下方的效率或累积武学查看该宠物的武学信息。
# 角色下线时宠物仍可继续修炼。
