D1 
h1 匹配评价
h2 系统入口
# 组队进行刷道、除暴、修行、十绝阵战斗时可对队友进行评价。
# 评价功能入口在组队界面或主界面组队面板点击队友弹出的菜单中。
h2 评价规则
# 进行评价后系统会根据队友在战斗中采取的操作进行判定，若情况属实则评价生效会对被评价者的评分产生影响，否则评价不生效。
# 差评生效时被评价者还将收到差评提醒邮件，若有开启推送还将收到推送。
# 为了避免一时失误即被多次差评，差评每小时最多生效一次，如果收到差评提醒还请道友们多多关注，及时调整自身自动战斗配置。
# 在进行组队匹配时，评分相近的角色将更容易被匹配到一起，即评分高的容易匹配到评分高的，评分低的容易匹配到评分低的。
# 积极组队进行战斗，一段时间内不再受到生效的差评则匹配评分会缓慢上升。
# 为了保证众位道友的匹配体验，还请组队挂机前多多关注自身的自动战斗配置情况哦。
